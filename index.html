##vue create xxx 创建的xxx文件无法删除问题

在我们利用vue脚手架，用vue create创建一个脚手架模板时，我们可以在里面写自己的组件。但是当我们不需要是，利用右键->删除时，会报错：有些文件需要管理员权限才能删除。由于里面的文件太多不知道怎么找到。

  • 解决办法：
    • 以管理员运行cmd ，切换到xxx文件夹的目录（cmd命令： cd 文件夹->切换到这个文件夹，cd 盘符: - >切换到该盘符）

这是我创建文件夹的目录。

输入：dir 查看里面的所有文件

  • 可以看到里面有很多文件夹，用：dir * 删除所有文件夹

    输入y后回车就删掉了所有文件。

  • 最后手动打开改文件夹，删除下面这两个文件：

    .git 文件夹
    node-modules 文件夹

  • 回到命令行输入：cd .. 切回到上一级目录 

然后输入：rd xxx就能删除文件夹了。
管理员命令是有权限删除所有文件夹，所以以后遇到这种问题都可以用这种方法。

mysql链接问题

发表于 2021-11-09 更新于 2021-11-12

本地之前有数据库，后来装了mysql8.0 导致本地连接不上去

2059 - Authentication plugin 'caching_ sha2_ password' 
cannot be loaded: 9X0000099g9頗

1.选择数据库mysql

mysql> use mysql
Database changed
mysql> select user,host,plugin,authentication_string from user;

2.更改加密方式

mysql> ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'password';
Query OK, 0 rows affected (0.35 sec)

3.刷新权限

mysql> FLUSH PRIVILEGES;
Query OK, 0 rows affected (0.28 sec)
